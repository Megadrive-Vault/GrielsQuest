Detailed Description

VDP Sprite support.

Author:
Stephane Dallongeville
Date:
08/2011

This unit provides methods to manipulate the VDP Sprites.
The Sega Genesis VDP can handle up to 80 simultanous sprites of 4x4 tiles (32x32 pixels).
